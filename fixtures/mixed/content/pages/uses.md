## What I use

A running list of the tools behind this site.

- **Generator:** colophon, building a static tree from Markdown.
- **Editor:** anything that writes plain text and frontmatter.
- **Hosting:** a CDN for HTML, object storage for assets and the search index.
- **Domain:** a placeholder like `example.com` while this is just a fixture.
