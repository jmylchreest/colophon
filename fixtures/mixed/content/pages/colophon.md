## About this site

This site is built with **colophon**, a themed Markdown blog generator with
pluggable publishers. Content is plain Markdown; pages and posts are rendered
into a static tree and deployed to object storage and a CDN.

- **Theme:** the active theme is selected per environment.
- **Search:** a fully static, sharded index — no server, no third party.
- **Feeds:** RSS, Atom, and JSON, generated at build time.

The source of truth is text files in a repository. Write it once, own it
forever, publish anywhere.
